# Morainet

Inspired by the geological term *moraine* — the enduring traces left by glaciers.

Morainet is a technology collective focused on software engineering, artificial intelligence, open source, and digital infrastructure.

We believe that meaningful technology is built layer by layer, contribution by contribution, forming ecosystems that endure beyond individual projects.

## Focus Areas

- Artificial Intelligence
- Mobile Development
- Open Source Software
- Developer Tools
- Cloud Infrastructure

## Philosophy

Build. Share. Evolve.
